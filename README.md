# MODERN MARISA LAB

Street Fighter 6「モダンマリーザ」の全技データベースと、判断練習を一体化した静的Webアプリです。

## 目的

技を読むだけで終わらず、対戦で起きた失敗を次の練習と実戦判断へ変換します。

1. 最近の負け方を選ぶ
2. 一つの判断技能へ絞る
3. 二択・ランダム問題へ回答する
4. 正答、見逃し、誤反応、入力違いを記録する
5. SF6トレーニングモードの設定例で実機練習する
6. 翌日テストと実戦ミッションへ進む
7. 各技が当たった条件から、次のコンボ・起き攻めを選ぶ

## 主な機能

- 全画面のコントローラー型UI
- 方向＋弱・中・強・SPによる技の逆引き
- 学習用のA弱・A中・A強専用ボタン
- アシストコンボ3種の役割比較
- 対空、インパクト、ヒット確認、近距離防御、確定反撃、アシスト選択の判断ドリル
- 二択10問、ランダム20問、翌日15問
- 正答、見逃し、誤反応、入力違い、判断違いの記録
- 合格課題の1日後、3日後、7日後、14日後レビュー
- 次の10試合で試す実戦ミッション
- LocalStorageによる端末内の進捗保存
- モダンで使用可能なマリーザ全48技の性能データ
- 各技の通常、カウンター、パニッシュカウンター、ラッシュ別ヒット後おすすめ
- 各条件につき最大3件のコンボ・起き攻め・位置継続案
- 火力、継続、簡単さ、ゲージ、難度による比較
- 推奨ルートごとの参照元と検証状態
- 状況別フィルターと実戦攻略デッキ
- PC／スマートフォン対応

## ヒット後おすすめの読み方

各技カードの「ヒット後おすすめ」を開き、状況タブを選びます。

- `通常`：通常ヒット時
- `カウンター`：通常より原則+2F
- `パニカン`：通常より原則+4F
- `ラッシュ`：ラッシュ通常技のヒット・ガード有利が原則+4F

候補は最大3件です。最適解が一つに絞れる場合は一件だけ表示します。

評価項目：

- 火力
- ヒット後の継続・起き攻め
- 入力の簡単さ
- ドライブゲージ
- SAゲージ
- 入力難度

検証ラベル：

- `資料照合`：公式数値と攻略資料を突き合わせた候補
- `距離依存`：密着、先端、立ち／しゃがみなどで成立が変わる
- `フレーム差分推定`：硬直差から成立候補を算出。ゲーム内で最終確認が必要
- `通常ルート共通`：専用リンクより通常ヒットと同じ安定ルートを優先

推定ルートを確認済みルートとして扱わない設計です。距離、持続当て、相手状態、画面端で変わるものは条件を明記します。

## 練習モードの操作

- 画面上部または下部の`DRILL`から開始
- `T`：練習モードを開く
- 矢印：方向入力
- `J`：弱
- `K`：中
- `L`：強
- `I`：SP
- `Q`：A弱
- `W`：A中
- `E`：A強
- `D`：インパクト返し
- `Esc`：練習モードを閉じる

ブラウザ内のドリルは状況認識と回答選択を扱います。実際の入力タイミングと成功可否は、各技能に表示される設定をSF6のトレーニングモードで再現して確認してください。

## ファイル構成

- `data-core.js`、`data-1.js`〜`data-6.js`：全48技の基本データ
- `followups-data.js`：条件別の推奨ルート、評価、出典、検証状態
- `followups-engine.js`：全技カードへの条件タブ・推奨ルート表示
- `followups.css`：ヒット後おすすめ比較UI
- `playbook-data.js`：実戦攻略カード
- `training-data.js`：技能、ドリル、問題、ゲーム内設定
- `training-engine.js`：診断、出題、採点、翌日テスト、実戦ミッション
- `controller-shell.js`：コントローラーUI
- `training-system.css`：練習画面
- `version.json`：公開版とデータ基準

## ローカル確認

ビルド不要です。ローカルサーバーを起動してください。

```bash
python -m http.server 8000
```

`http://localhost:8000/`を開きます。

## 自動検査

GitHub Pages公開前に次を検査します。

- JavaScript構文
- 基本技データ48件とID重複
- 条件別データの対象48件との完全一致
- `normal / counter / punish / rush`の4条件
- 一条件あたり1〜3件という上限
- 存在しない技ID、条件ID、出典IDの参照
- 推奨が未登録の技に通常ヒット後のフォールバックがあること
- CSSの括弧対応
- 公開HTMLの読み込み順とバージョン

## データ基準

技データは2026年3月17日のバランス調整後を基準にしています。アップデート後はCAPCOM公式フレームデータを優先して更新してください。

コンボダメージは始動補正、ドライブゲージ、SAゲージ、CA、画面位置、簡易入力、相手状態で変動します。数値が確定できないものは「未計測」または概算として表示します。

アシスト中コンボなど条件が未確定の内容は、練習モードでは正確なダメージ暗記に使わず、ヒット時に続けるか、ガード時に止めるかという判断練習に限定しています。
