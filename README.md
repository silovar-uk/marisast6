# MODERN MARISA LAB

Street Fighter 6「モダンマリーザ」の技データベースと、判断練習を一体化した静的Webアプリです。

## 目的

技を読むだけで終わらず、対戦で起きた失敗を次の練習へ変換します。

1. 最近の負け方を選ぶ
2. 一つの判断技能へ絞る
3. 二択・ランダム問題へ回答する
4. 正答、見逃し、誤反応、入力違いを記録する
5. SF6トレーニングモードの設定例で実機練習する
6. 翌日テストと実戦ミッションへ進む

## 主な機能

- 全画面のコントローラー型UI
- 方向＋弱・中・強・SPによる技の逆引き
- 学習用のA弱・A中・A強専用ボタン
- アシストコンボ3種の役割比較
- 対空、インパクト、ヒット確認、近距離防御、確定反撃、アシスト選択の判断ドリル
- 二択10問、ランダム20問、翌日15問
- 正答、見逃し、誤反応、入力違い、判断違いの記録
- 合格課題の1日後、3日後、7日後、14日後レビュー
- 次の10試合で試す実戦ミッション
- LocalStorageによる端末内の進捗保存
- 全技のダメージ、発生、持続、硬直、ヒット／ガード硬直差
- 状況別フィルターと実戦攻略デッキ
- PC／スマートフォン対応

## 練習モードの操作

- 画面上部または下部の`DRILL`から開始
- `T`：練習モードを開く
- 矢印：方向入力
- `J`：弱
- `K`：中
- `L`：強
- `I`：SP
- `Q`：A弱
- `W`：A中
- `E`：A強
- `D`：インパクト返し
- `Esc`：練習モードを閉じる

ブラウザ内のドリルは状況認識と回答選択を扱います。実際の入力タイミングと成功可否は、各技能に表示される設定をSF6のトレーニングモードで再現して確認してください。

## ファイル構成

- `data-core.js`、`data-1.js`〜`data-6.js`：技データ
- `playbook-data.js`：実戦攻略カード
- `training-data.js`：技能、ドリル、問題、ゲーム内設定
- `training-engine.js`：診断、出題、採点、翌日テスト、実戦ミッション
- `controller-shell.js`：コントローラーUI
- `training-system.css`：練習画面
- `version.json`：公開版とデータ基準

## ローカル確認

ビルド不要です。ローカルサーバーを起動してください。

```bash
python -m http.server 8000
```

`http://localhost:8000/`を開きます。

## データ基準

技データは2026年3月17日のバランス調整後を基準にしています。アップデート後はCAPCOM公式フレームデータを優先して更新してください。

アシスト中コンボなど条件が未確定の内容は、練習モードでは正確なダメージ暗記に使わず、ヒット時に続けるか、ガード時に止めるかという判断練習に限定しています。
