# MODERN MARISA LAB

Street Fighter 6「モダンマリーザ」の全技を、入力・性能・使いどころ・ヒット後の行動から逆引きできる静的Webアプリです。

## 特徴

- N・矢印を使った、テンキー不要の入力表記
- モダンで使用できる通常技、特殊技、ターゲットコンボ、ジャンプ技、必殺技、SA、投げを収録
- ダメージ、発生、持続、硬直、全体、ヒット／ガード硬直差
- 「対空」「牽制」「差し返し」「起き攻め」など状況別フィルター
- 各技から次につながる技・代表的な行動を掲載
- モダン簡易入力の80％ダメージを併記
- PC／スマートフォン対応

## ローカル確認

ビルド不要です。`index.html` をブラウザで開くか、ローカルサーバーを起動してください。

```bash
python -m http.server 8000
```

## データ基準

2026年3月17日のバランス調整後を基準に作成しています。アップデート後はCAPCOM公式フレームデータを優先して更新してください。

非公式のファンメイド攻略サイトです。

## 性能評価

発生・持続・硬直・ダメージ・ヒット差・ガード差を、絶対値の目安で「◎ 強い」から「× 大きな隙」まで色分けしています。これは技全体の強さではなく、各数値の性質を理解するための補助表示です。

## 状況別ガイド

地上戦、対空、守り、弾対策、起き攻め、コンボ、画面端、ゲージ判断の8分野に分け、各5状況・合計40状況から推奨技を絞り込めます。
